# cubingitaly.org
**Requirements**

 - Node
 - Mysql

**Install**

    git clone https://github.com/CubingItaly/cubingitaly.org.git && cd cubingitaly.org
    npm install
    
**DB setup**
Database config can be found in ormconfig.json

    mysql -u username -p
    create database ci_web_db

**Run in development mode mode**

    npm run start
